Detailed Description

Sequence of optical parts with associated ray pack, raytracing application and ray states saving.

lseq is the top-level API class for the RobinTrace raytracer. It holds the definition of a sequence of lparts, rays to apply the sequence to, and buffers for saving the ray states along the way.
